# Webスクレイピングサンプル2

## 概要
- requests を使ってソースの取得
- BeautifulSoup を使って必要な要素を取得
- extract() で不要な部分を削除
- 取得した情報を辞書型で保存
- pandas を使ってデータフレーム化
- 項目を並べ替えてCSV形式で保存

## 開発環境
- Windows 10 Home
- Python 3.9.4
- VS Code 1.55.1

## 仮想環境の作成
```
py-m venv .venv
```

## 仮想環境の有効化(Windows)
```
.venv\Scripts\activate
```

## 必要なパッケージのインストール
```
python -m pip install -r requirements.txt
```
